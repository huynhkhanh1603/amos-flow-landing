Today you choose one: a rigid workflow that breaks when things change, or a free-roaming agent you cannot fully trust.

Supervisor agent

Runs the workflow from the outside. It builds the workflow, triggers each run, monitors its health, and heals it when a step fails.

Task agent

Lives inside a single step. It handles the unstable or ambiguous actions that fixed rules cannot, then hands control straight back.

Two specialized agents, not one. Each is tuned for its own job.
